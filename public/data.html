Data
What data is being collected?
Depending on the exact site different data may be collected. This data almost always includes IP Address, country of origin (provided by Cloudflare), User-Agent, time of access, the accessed URL. Additionally some API usage is logged. This includes any login attempts, account modifications, account deletions, and more. When using a service that requires a login the username / user id is also logged. Any errors that might occur during usage of a service may also be logged for the purpose of improving the service.
